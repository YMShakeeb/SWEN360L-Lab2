# Green Campus Management
My project idea is improving the campus suatainability, and making it more green, focusing on saving energy.
Technologies or tools used are HTML, Microsoft Access.

To use the projects, usres must follow the steps below:
- downlaod VSCode APplication
- open Vscode
- download external Server in the VScode application
- open the link shared to access my github (recommended to use website not application).
- downloaded the files listed.
- in Vscode click Open
- open files (do not open wwith the main files, only with sub files, to access sub files must click on -> github -> main file name-> sub file name).
- click on index.html code page
- Click on (Go Live) on the bottom left of VScode page - (Must only open server on same page of index.html).
- Sucessfuly Accessed the Project.
